# Описание игры

Вы - отважный исследователь в древнем лабиринте, полном хитрых коридоров, скрытых сокровищ и загадок, 
которые проверят вашу смекалку. Это не просто игра — это ваша собственная текстовая adventure игра 
"Лабиринт сокровищ", где каждый шаг может привести к новому открытию или опасности.

# Особенности игры:

Исследуйте таинственные комнаты лабиринта

Собирайте древние артефакты и предметы

Разгадывайте загадки и головоломки

Используйте предметы для преодоления препятствий

Найдите главное сокровище лабиринта!

Быстрый старт

Установка и запуск через Makefile

# Клонирование репозитория

git clone https://github.com/nikolayzhmurov88-star/project1_Zhmurov_Nikolai_M25-555.git
cd project1_Zhmurov_Nikolai_M25-555

# Установка и запуск игры

make install

make project

# Доступные команды Makefile

# Команда	               Описание

make install	           Установить зависимости через Poetry
make project	           Запустить игру
make build	               Собрать пакет для распространения
make publish	           Тестовый запуск публикации (dry-run)
make package-install	   Установить собранный пакет
make lint	               Проверить код с помощью ruff

# Команды игры

Вот основные команды, которые вы можете использовать во время игры:

# Команда	                                    Описание

north / south / east / west	                    Перейти в соответствующем направлении
look	                                        Осмотреть текущую комнату
take <item>	                                    Поднять предмет (например: take ключ)
use <item>	                                    Использовать предмет из инвентаря
inventory	                                    Показать инвентарь
solve	                                        Попытаться решить загадку в комнате
quit	                                        Выйти из игры
help	                                        Показать список команд
